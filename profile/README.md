Hello! I love doing all sorts of things with code. Here are some of the coolest things I've made or contributed to:

## [Pandapip1/jekyll-label-action](https://github.com/Pandapip1/jekyll-label-action)

A GitHub action that helps organize PRs with Jekyll websites by programmatically adding labels depending on the Front Matter of changed pages.

Used by [ethereum/EIPs](https://github.com/ethereum/EIPs).

## [OpenSchooling/AccentPress](https://github.com/OpenSchooling/AccentPress)

A [Chrome Extension](https://chrome.google.com/webstore/detail/accent-press/nfcdcdoegfnidkeldipgmhbabmndlhbf) to easily type accents on keyboards that don't support them. 

Used by my former classmates in French class.

## [Pandapip1/hclustering](https://github.com/Pandapip1/hclustering)

A proof of concept of a simple heirarchal clustering algorithm I made.

Used by me, primarily.

## [Pandapip1/mineflayer-swarm](https://github.com/Pandapip1/mineflayer-swarm)

An [NPM package](https://www.npmjs.com/package/mineflayer-swarm) to control an army of bots in Minecraft.

I don't know who uses it.
